## RandomUser caseira:

Entregável: uma API que retorna um usuário aleatório ao receber uma requisição GET no "/". As
informações dos usuários devem conter, pelo menos:
. Primeiro nome
. Segundo nome
. Gênero
. Data de nascimento
. Email
. Senha

## Technologies
The following technologies was used to complete the challenge:

. Node.js
. Express
. Faker
