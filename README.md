## RandomUser caseira:

Entregável: uma API que retorna um usuário aleatório ao receber uma requisição GET no "/". As
informações dos usuários devem conter, pelo menos:
- Primeiro nome
- Segundo nome
- Gênero
- Data de nascimento
- Email
- Senha

## Tecnologias
Foram usadas:

- Node.js
- Express
- Faker
